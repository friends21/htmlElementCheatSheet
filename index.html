HTML Elements Cheat Sheet

HTML Element Tags


Tag Name Description
<li> list item Creates items inside ordered and unordered lists.
<ol> ordered list Create lists that orders first item as 1, second as 2, and so on.
<ul> unordered list Creates unordered lists with bullet points.
<video> video Embeds media player for video playback.
<em> emphasis Emphasises text by italicising it.
<div> division Divides HTML content into sections.
<br> line break Creates a line break in text.
<img> image Embeds images in HTML files.
<h1>-<h6> heading Ordered from highest level, <h1>, to lowest level, <h6>.
<p> paragraph Contains and displays a block of text.
<body> body HTML content displayed on web browsers is held inside the <body> tag.
<span> span Inline container for text and can be used to group text for styling purposes.
<strong> strong Makes text bold.
<a> anchor The anchor element is used to create hyperlinks in an HTML document.
<head> head Contains site information that isn't displayed, such as metadata.
<html> html All content and structure in an HTML document is contained in the <html> tag.
<title> title The <title> element is conatined in the <head> tag, and is used to display text in a browser's tab.

HTML Attributes


Attribute Name Description
id id Unique id attributes can be assigned to differnet HTML elements in order to differentiate between them and make them accessible to CSS and JavaScript files.
alt alternative text Displays alternative text if media fails to render.
<target> target When set to value of "_blank", opens link in a new tab or window.
